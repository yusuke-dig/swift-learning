# Day02 Review: 基本型と型推論

## 課題1: celsiusToFahrenheit

```swift
// 実装
return celsius * 1.8 + 32
```

**良かった点**: 公式をそのままシンプルに1行で書けています。Double同士の演算なので型変換も不要で、明快な実装です。

---

## 課題2: isLongWord

```swift
// 実装
return word.count >= 6
```

**良かった点**: `String.count` で文字数を取得し、比較演算子だけで完結しています。これが標準的な書き方です。

**より良い書き方（補足）**: 絵文字や一部のUnicode文字は `.count` でも正しく1文字として数えられます（Swiftの `Character` はUnicodeクラスターを1文字扱いするため）。バイト数でなく文字数で判定できているのが Swift らしい点です。

---

## 課題3: absolute

```swift
// 実装
return n >= 0 ? n : -n
```

**良かった点**: 三項演算子を使って1行でスッキリまとめられています。`abs()` を使わずに条件分岐で実装するという課題の意図を正確に捉えています。

**別の書き方（参考）**:
```swift
// if文で書くと意図がより明示的
if n >= 0 { return n } else { return -n }
```

どちらも正解ですが、慣れてきたら三項演算子の方が簡潔でよく使われます。
